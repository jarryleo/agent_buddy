# agent_buddy

Agent Buddy

目标:跨端智能体基座

### UI布局:
1.无论哪端,都采用手机样式.
2.APP启动后进入聊天页面.左上角设置按钮,点击进入设置页面.
3.设置页面,顶部tab:Provider,Role,Tools,Skill
Provider:模型提供商列表,可以新增,编辑,删除,启用.功能见下面功能详情.
Role:AI角色扮演列表,展示角色列表,可新增,编辑,删除,启用.
Tools:AI智能体工具列表,管理内置工具,先实现Fetch Web,可以让AI获取网址内容.
Skill:技能列表,与现有市面上技能兼容.
~~ UI样式采用IOS设计样式,简约素雅风格 ~~

### 功能:
#### 模型提供商:
1.先实现设置AI模型提供商页面,需要支持OpenAI和Anthropic协议.
2.可以添加多个提供商,添加页面输入名称, url, API key,需要根据协议自动补全path,有获取模型列表按钮.有测试连接按钮.

#### 聊天首页:
1.实现对话聊天页面,支持流式输出,支持不同模型的思考模式,AI聊天气泡5行思考流式展示,回复内容后,思考部分可展开至10行,上下滚动.
2.底部输入对话框最多展示10行,超过10行后可滚动.右边发送按钮与输入框底部对齐,整个输入框部分采用悬浮条设计.

#### 设置页面:
见上面UI布局.
